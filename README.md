# Proyecto del Sprint 7 - Análisis de Vehículos

Este proyecto consiste en una aplicación web interactiva desarrollada con **Streamlit**, diseñada para explorar y visualizar un conjunto de datos de anuncios de venta de coches en los EE. UU.

## Funcionalidades
- Visualización de la distribución del kilometraje (odómetro) mediante histogramas.
- Análisis de la relación entre el año del modelo y el precio a través de gráficos de dispersión.
- Interfaz interactiva con botones para generar gráficos bajo demanda.

## Tecnologías utilizadas
- Python
- Pandas
- Plotly Express
- Streamlit
